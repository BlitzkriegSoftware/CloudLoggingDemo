# Cloud Logging Demo #
A demo of how to log in the cloud

This works in:
* PCF
* Kubernetes (K8s)
* Azure
* AWS EC2
* GCP GCE

## Set up your cloud ##

1. Create a place to deploy
2. If needed set up collecting STDOUT/STDERR
3. Deploy
4. View logs using cloud mechanism
